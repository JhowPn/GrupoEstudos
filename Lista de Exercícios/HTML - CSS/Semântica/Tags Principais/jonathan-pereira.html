"font family" essa tag designara a fonte que a letra irá aparecer e existem dois tipos de "font family" a primeira (nome de familia) e (familia generica) a primeira citada é com se fosse a oficial e a segunda citada é caso o navegador não abra a oficial, usará a fonte familia generica para substituir por uma fonte reserva.

"font weight" define diferentes espessuras para 3 parágrafos.

"text decoration" é usada para remover o sublinhado dos links.

"padding" essa tag é usada para o preenchimento de um elemento em uma borda definida.
